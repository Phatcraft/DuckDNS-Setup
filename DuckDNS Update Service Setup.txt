1. Thêm file /etc/systemd/system/DuckDNS.service

2. Thêm nội dung file

[Unit]
Description=DuckDNS Update Service
After=network-online.target
Wants=network-online.target

[Service]
Type=oneshot
ExecStart=/bin/bash /home/<username>/duckdns/duck.sh
RemainAfterExit=no

[Install]
WantedBy=multi-user.target

3. Sau khi tạo file, reload daemon, start và enable DuckDNS service
$ sudo systemctl daemon-reload
$ sudo systemctl enable DuckDNS
$ sudo systemctl start DuckDNS
