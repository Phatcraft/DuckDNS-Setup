# DuckDNS Setup

## Về repository
Đây là repository hướng dẫn setup DuckDNS trên server và cấp Let's Encrypt certificate cho DuckDNS domain.<br>
Bạn có thể xem hướng dẫn này nhằm setup DuckDNS domain cho server của bạn.

> [!WARNING]
> *Lưu ý: Hướng dẫn được tạo ra bằng chatGPT + thử nghiệm trên server ảo nên một số thứ trong hướng dẫn không hoàn toàn chính xác*
