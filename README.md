# DuckDNS Setup
Đây là repository chứa các hướng dẫn đối với DuckDNS, bao gồm
+ Setup cập nhật địa chỉ IP server lên DuckDNS server
+ Setup service để thực thi script ngay sau khi chạy server
+ Setup Certbot để cấp certificate cho DuckDNS domain

*Note: Các hướng dẫn trên được tạo ra nhờ ChatGPT + thực nghiệm trên máy ảo*
